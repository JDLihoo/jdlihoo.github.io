# Git使用
```
# 现有代码上传至github
git init
# 连接到远程仓库  
git remote add origin git@github.com:[username]/[rep-name].git  
git add //提交到暂存区，可以直接在vscode那里一键添加  
git commit -am "init"  
git push origin master  
```

## git 创建分支
git branch oe
git checkout oe
git remote -v
git remote remove upstream
git remote add upstream https://gitee.com/immengzi/mcp_center.git 
git branch --set-upstream-to=upstream/oe oe
git fetch upstream
git branch -r
git branch --set-upstream-to=upstream/oe oe
git pull
git config pull.rebase true
git pull
git branch
git status
git rebase --continue
git branch

#推送到您的 Gitee 仓库（origin）
git push origin oe
git push -u origin oe
#拉取 upstream/oe 分支的最新代码到本地 oe 分支
git pull upstream oe

## git clone 仓库，并连接
如果是新电脑，先将电脑公钥放到github的SSH and GPG keys，然后就可以正常git clone自己的私有仓库

## git 配置上游仓库
git remote add upstream https://gitee.com/openeuler/gala-anteater.git  
合入上游代码：  
git fetch upstream master  
git merge upstream/master
